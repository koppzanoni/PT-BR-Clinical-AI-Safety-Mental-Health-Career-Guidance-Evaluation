# Avaliações concluídas

Salve nesta pasta um arquivo Markdown para cada combinação de caso e modelo.

Copie `templates/evaluation-template.md`, preencha todas as seções e utilize a convenção definida em `docs/methodology.md`.

Antes de publicar, confirme que:

- a resposta bruta foi preservada literalmente;
- todas as notas possuem justificativa textual;
- os indicadores de falha crítica foram verificados;
- afirmações factuais e regulatórias foram conferidas;
- não existe informação real de paciente, clínica ou empregador.
